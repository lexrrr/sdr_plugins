This has started as very simple tetra receiver for sdr# based on Telive on Windows by cURLy bOi
================================

sdr# plugin magic line:
<add key="Tetra" value="SDRSharp.Tetra.TetraPlugin,SDRSharp.Tetra"/>

Required to run:
Visual C++ Redistributable Packages for Visual Studio 2013
https://www.microsoft.com/en-eg/download/details.aspx?id=40784

Install:
---------
1) Copy ddl's to sdr# directory
2) Add magic line to Plugins.xml

Use:
---------
1) Run sdr#
